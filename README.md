## Terminal Tetris

Before you run the game:
- Enter your own directory path in `tetris.py` (and uncomment it)

Or:
- Open your directory in terminal
- Run this command:

      python tetris.py

Requirements:
- Python `keyboard` package must be installed
- If not, use this command:

      pip install keyboard
